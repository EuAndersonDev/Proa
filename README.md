## 🚀 Tecnologias e Conteúdos Abordados

Ao longo do curso, estou estudando e praticando:

* **HTML5** – estruturação de páginas web
* **CSS3** – estilização, layouts e responsividade
* **JavaScript** – lógica, manipulação do DOM e interatividade
* **Lógica de Programação** – base para resolução de problemas
* **Programação Orientada a Objetos (POO)** – conceitos como classes, objetos, herança e encapsulamento
* **Java** – fundamentos da linguagem, POO e desenvolvimento de aplicações
* **Projetos Pequenos** – aplicações práticas para fixação do conteúdo
